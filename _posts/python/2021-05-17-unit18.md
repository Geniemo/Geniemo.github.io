## break
```python
i = 0
while True:
    print(i)
    i += 1
    if i == 10:  # i가 10이면 while문 탈출
        break

for i in range(100000):
    print(i)
    if i == 10:  # i가 10이면 for문 탈출
        break
```

## continue
```python
for i in range(20):  # 0부터 19까지
    if i % 2 == 0:  # 짝수면
        continue  # 아래 코드 실행하지 않고 넘어감
    print(i)

i = 0
while i < 20:  # i가 20보다 작을 때 반복
    i += 1
    if i % 2 != 0:  # 홀수면
        continue  # 아래 코드 실행하지 않고 넘어감
    print(i)
```
