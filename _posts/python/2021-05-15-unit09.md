## 문자열 사용법

문자열은 ''나 ""안에 묶어서 사용<br>
여러줄로 된 문자열은 ''' '''나 """ """ 안에 묶어서 사용
```python
str1 = 'str1'
str2 = "str2"
str3 = '''str3_1
str3_2
str3_3'''
print(str1)  # str1
print(str2)  # str2
print(str3)
# str3_1
# str3_2
# str3_3
```

문자열 안에 따옴표를 넣을 때<br>
"" 안에 " 를 넣으려면 \"<br>
'' 안에 ' 를 넣으려면 \'<br>
''' ''', """ """ 안에는 따옴표가 문제없이 다 넣어진다.
```python
print("'''  \"")  # '''  "
print('"""  \'')  # """  '
print('''  ' ''')  #   ' 
print("""  "  """)  #   "  
```
